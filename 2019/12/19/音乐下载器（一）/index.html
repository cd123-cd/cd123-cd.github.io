音乐下载器（一）

今天，不对应该是昨天，为了六行代码差点没搞死，以至于现在凌晨两点四十还在这查百度，哈哈骗人的啦，失恋了睡不着，就打打代码吧
这个程序是跟着b站学的https://www.bilibili.com/video/av77225642?t=935
出现三个问题
报错1：
FileNotFoundError: [WinError 2] 系统找不到指定的文件。
解决办法找到lib中的subprocess.py文件
把 init函数中的shell=False改成shell=True
我也是百度查的具体参考https://blog.csdn.net/qq_43403025/article/details/100023633
我的源代码此时是这样的

#自动化框架
from selenium import webdriver
import time
driver = webdriver.Chrome()
url = r’https:\y.qq.com\portal\search.html#page=1&searchid=1&remoteplace=txt.yqq.top&t=song&w=%E4%BC%A4%E5%BF%83%E5%A4%AA%E5%B9%B3%E6%B4%8B’
driver.get(url)
报错二：
selenium.common.exceptions.WebDriverException: Message: ‘chromedriver’ executable needs to be in PATH. Please see https://sites.google.com/a/chromium.org/chromedriver/home
这是因为我没装驱动，此时你要做的是查一下你的谷歌版本，然后下载对应驱动，保存在chrome下就可以了
参考https://blog.csdn.net/qq_42711815/article/details/89558107
源文件和上面一样没改
你以为这样就结束了吗？no
报错三：
selenium.common.exceptions.WebDriverException: Message: Service chromedriver unexpectedly exited. Status code was: 1
我把文件刚下好的驱动路径放进去就可以了
源代码：
from selenium import webdriver
import time
driver = webdriver.Chrome(r’C:\Users\LEGION\AppData\Local\Programs\Python\Python37-32\Lib\site-packages\selenium\webdriver\chrome\chromedriver.exe’)
url = r’https:\y.qq.com\portal\search.html#page=1&searchid=1&remoteplace=txt.yqq.top&t=song&w=%E4%BC%A4%E5%BF%83%E5%A4%AA%E5%B9%B3%E6%B4%8B’
driver.get(url)
友情提示1.路径用反斜杠 2.记得路径前加r要不然编码错误
